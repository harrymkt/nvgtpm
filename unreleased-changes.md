## New in Version 0.0.3
- Added `home` subcommand to `bucket` command that allows you to open the URL (if available) of a given bucket in your browser.
- Improved `decl` command. It now use either `main.nvgt` or `<package_name>.nvgt`.
- Added `-c` or `--copy` option to `decl` command to copy the declaration syntax to the clipboard.
- Removed `cleanup` command.
- Added `cache` command. Commands such as `rm` to remove, are available at `cache <command>`.
